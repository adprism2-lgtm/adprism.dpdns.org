## Quick-reference: what the numbers mean

| Metric | Band | Read |
| --- | --- | --- |
| Volume | <100 niche · 100–1K small · 1K–10K solid | Higher = bigger audience |
| KD% | <30 realistic for a newer site · 30–50 needs good content + time · 50–70 ambitious · 70+ big-site territory | Lower = easier to rank |
| Intent | Commercial / Transactional = buyers | Prioritize these for booking pages |

**Priority rule of thumb:** target keywords that are **Commercial intent + decent volume + lower KD** first. Map each to the matching page:

| Keyword theme | Page to optimize |
| --- | --- |
| videographer / video production / event videography karachi | `/videography-karachi` |
| corporate video / documentary karachi | Films section + `/videography-karachi` |
| exhibition stall coverage karachi | Exhibitions section |
| product / corporate photographer karachi | Portfolio + Services |
